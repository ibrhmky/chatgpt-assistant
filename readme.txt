WooDev AI Content Generator is an AI-powered chat assistant for your WordPress website.

== Description ==

WooDev AI Content Generator is a cutting-edge AI-powered chat assistant for your WordPress website, revolutionizing the way you interact with your users. Leveraging the advanced capabilities of OpenAI's GPT-3 technology, this plugin transforms your website into an interactive hub where users can engage in meaningful conversations, receive instant responses, and access relevant information.

= Key Features =

- **Intelligent Conversations:** The plugin's chat assistant utilizes GPT-3 to understand user input, providing human-like responses that make interactions feel natural and engaging.

- **Expertise Modes:** Choose from various expertise modes, including Technology, Art, Marketing, and more. Each mode tailors the assistant's responses to match the chosen domain, ensuring relevant and accurate information delivery.

- **Bulk Input Processing:** Effortlessly process multiple messages at once using the bulk input mode. The assistant processes each message and provides responses in a sequential and efficient manner.

- **Seamless Post Integration:** Easily integrate AI-generated content into your posts. The assistant can create, refine, and suggest content for your articles, saving you time and enhancing your content quality.

- **Customizable Appearance:** Customize the chat assistant's appearance and behavior to match your website's design and user experience.

== Installation ==

1. Upload the `chatgpt-assistant` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your OpenAI API key in the plugin settings.
4. Add the chat assistant to your website using the provided shortcode or widget.
